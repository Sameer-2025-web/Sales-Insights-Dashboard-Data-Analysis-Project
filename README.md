# Sales Insights Dashboard

## *Objective*
The primary goal of this project is to create an interactive and visually appealing dashboard that allows stakeholders to:  
- Monitor overall sales performance  
- Analyze sales trends across regions and products  
- Identify top-performing products and regions  
- Make data-driven business decisions  


## *Project Overview*
This project demonstrates end-to-end Power BI skills, including:  
- Importing and cleaning data using Power Query  
- Building relationships and data models  
- Writing DAX calculations for KPIs, growth %, rankings, and YoY analysis  
- Designing interactive dashboards with slicers, charts, and visualizations  



## *Steps Followed*
1. *Data Collection:*  
   - Generated a sample sales dataset including columns for region, product, sales, quantity, and profit.  

2. *Data Cleaning & Transformation:*  
   - removed duplicates, rename columns, format data types, and handle missing values.  

3. *Data Modeling:*  
   - Created relationships between tables and designed a data model suitable for reporting and analysis.  

4. *DAX Calculations:*  
   - Created calculated measures for Total Sales, Total Profit, Growth %, Ranking, and Year-over-Year comparisons.  

5. *Dashboard Design:*  
   - Designed interactive visualizations using line charts, bar charts, KPI cards, maps, and slicers.  

6. *Validation & Testing:*  
   - Verified data consistency, filters, and calculated metrics.  

7. *Documentation:*  
   - Captured screenshots and prepared this README for portfolio and GitHub presentation.

---

## *Key Features*
- *Dynamic KPIs:* Real-time monitoring of revenue, profit, and sales trends  
- *Interactive Filters:* Slice data by region, product category, and time period  
- *Custom DAX Measures:* Growth %, ranking, and year-over-year analysis  
- *Visual Insights:* Line charts, bar charts, and maps for detailed sales analysis  
- *User-Friendly Design:* Clear layout for easy interpretation of data
